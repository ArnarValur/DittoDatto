Working with the APIs

---

All content that is not user created, such as company type names, can be translated to any of our supported languages via the **Accept-Language** header.

| Language | Accept-Language |
| --- | --- |
| English | en |
| Icelandic | is |
| Portuguese | pt |
| Czech | cs |
| Spanish | es |
| French | fr |
| Polish | pl |

If no language is specified via the Accept-Language header, we default to English.[Point In Time Properties](https://docs.noona.is/docs/working-with-the-apis/point-in-time-properties)
