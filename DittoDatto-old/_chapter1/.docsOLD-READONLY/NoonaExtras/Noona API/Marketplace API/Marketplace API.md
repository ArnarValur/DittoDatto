The Noona Marketplace API enables you to build customer-facing booking experiences. It provides endpoints for customers to discover service providers, view available time slots, book appointments, manage payments, and more.

## Key Features

The Marketplace API provides access to:

- **Discovery & Search** - Browse and search for companies, services, and employees
- **Booking Management** - Create, view, and manage appointments and events
- **Time Slots & Availability** - Check real-time availability and reserve time slots
- **Payments** - Process payments and manage payment methods
- **User Management** - Customer authentication, profiles, and preferences
- **Vouchers & Offers** - Create and redeem vouchers and special offers
- **Waitlists** - Join and manage waitlist entries

Browse the sidebar to explore all available endpoints, organized by resource type:

- **Marketplace Companies** - Access company profiles and opening hours
- **Event Types** - View available services and their details
- **Time Slots** - Check availability and make reservations
- **Marketplace Events** - Create and manage customer bookings
- **Payments** - Handle payment processing
- **Marketplace User** - Manage customer accounts
- **Vouchers** - Create and manage vouchers
