Reminders are configured for automated reminders to be sent to customers before events (appointments).

Reminders can be configured to send via SMS, with customizable timing (e.g., 24 hours before the event) and content.

Reminders can be scoped to specific employees or event types, or applied company-wide. SMS content can be translated into multiple languages.[List all notifications GET](https://docs.noona.is/docs/hq/push-notifications/ListPushNotifications)
